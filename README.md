# Adult Income Prediction
PROJECT OVERVIEW : 
The objective of this project is to build a predictive model that determines whether an individual earns more than $50,000 per year based on a variety of demographic and work-related attributes.
TOOLS USED
•	MS Excel
•	Power BI.
DATASET
•	Source:https://www.kaggle.com/datasets/mosapabdelghany/adult-income-prediction-dataset
•	Dataset contains contain 15 columns with mix of both categorical and numerical features such as age, education, occupation, race, final weight, income etc.
STEPS FOLLOWED
•	Cleaned data in Excel (handled missing values, formatted columns etc.)
•	Imported cleaned data into Power BI.
•	Changed datatypes accordingly.
•	Created calculated columns, measures and tables.
•	Built dashboards using charts, cards, slicers etc.
KEY INSIGHTS
•	The target income is over $50,000 per year. 
•	It is mainly achieved by individuals aged 40–49.
•	These individuals work in the private sector.
•	They hold a bachelor's degree.
•	Their occupation is executive or managerial.
•	They belong to the white racial group.
